# Project 2: Customer Value Segmentation (Excel)

## Overview
This project analyzes women's clothing e-commerce review data.
Customers were segmented into High, Medium, and Low value groups based on positive feedback count.

## Tools Used
- Microsoft Excel
- IF logic
- COUNTIF
- Pivot Tables
- Dashboard Design

## Process
- Cleaned raw data
- Standardized text values
- Created customer value segments
- Built pivot tables for analysis
- Designed a dashboard for insights

## Key Insights
- Low value customers represent the largest segment
- Tops and Dresses categories have strong high-value customer presence
- Segment distribution varies significantly across product classes
